# Hospital Management System <a name="about-project"></a>

The Hospital Management System is a web application developed using Ruby on Rails to manage patient information, appointments, medical records, and more. It allows hospital staff to efficiently manage patient data, track appointments, and provide seamless healthcare services.


## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- <a href="https://www.postgresql.org/">PostgreSQL</a>
- <a href="https://rubyonrails.org/">Ruby on Rails</a>

<!-- Features -->

### Key Features <a name="key-features"></a>

Key features of the application.

- **Patient Registration**: Add and manage patient details.
- **Appointment Scheduling**: Schedule and track patient appointments.
- **Doctor Dashboard**: View and manage patient medical history.
- **Admin Dashboard**: Administrative control to manage hospital staff and patient data.
- **Authentication**: Secure login and user authentication.
- **Search Functionality**: Search patients by name, ID, or condition.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- PostgreSQL.
- Code Editor.
- Ruby On Rails.
  

### Setup

Clone this repository to your desired folder:

```sh
  cd Hospital-management-system
  gh repo clone https://github.com/fatmahussein/hospital-management-system.git
```

### Install

- Install <a href="https://www.postgresql.org/">PostgreSQL</a>.
- Install <a href="https://rubyonrails.org/">Ruby On Rails</a>.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Run Test
To run the testing, execute the following command:
```
rspec spec
```
